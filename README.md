# Erdos Problems

Lean 4 formalizations for problems from
[erdosproblems.com](https://www.erdosproblems.com/).

## Included Problems

- `905`: Every graph with `n` vertices and more than `n^2 / 4` edges contains
  an edge lying in at least `n / 6` triangles.

## Layout

- `XXX/README.md`: short problem note.
- `ErdosProblems/XXX.lean`: Lean formalization.
